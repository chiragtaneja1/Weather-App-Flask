## Installation

- Clone the Repository
```bash
https://github.com/chiragtaneja915/Weather-App-Flask.git
```
- Navigate to the folder
```bash
  cd Weather-App-Flask
```
  - Install the required packages
```
pip install -r requirements.txt
```
## Run App
```bash
python app.py
```
## View Link
```bash
http://127.0.0.1:5000
```
